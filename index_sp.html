Robotic Process Automation
¿Qué es?

Se trata de un conjunto de tecnologías que permiten la automatización de un amplio rango de actividades manuales repetitivas dentro de las empresas sin la necesidad de una programación compleja. Puede aplicarse en procesos repetitivos de finanzas, alta de clientes, liquidación de impuestos, contabilidad, recursos humanos, gestión de la cadena de suministro, servicio al cliente, entre otros.

Casos de Éxito

Liquidación de Impuestos

Se realizó la programación de diversos robots que realizan la preparación de papeles de trabajo, generación de archivos de liquidación y presentación en la web del ente gubernamental de impuestos para mas de mil clientes. Antes de esta implementación este proceso era realizado por 15 personas cada mes. Hoy este equipo realiza actividades de mayor valor y el proceso se encuentra 100% automatizado.

Conciliaciones Bancarias

Se construyó un robot que permite descargar información de movimientos contables de un sistema SAP, descargar movimientos de home banking de 2 bancos y realizar una conciliación automática. Además informa vía mail los movimientos descuadrados y realiza un informe para un área centralizada.
